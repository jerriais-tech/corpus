Lé Cadet mantchi

Au run d'eune fûsi-machinne, des flieurs;
la pliant'tie au run d'pathades;
chouaîsi des bordeûthes au run d'bèrrexes;
mither eune frouque au run d'eune arme;
au run d'eune achie d'boulets, arrouôser atout d'l'ieau du ciel;
s'mer d'la graine au run d'entèrrer des machacrés;
au run d'la baïonnette, des belles-toutes-nues;
saluer l'èrnouvé au run d'un cap'taine -
dév'nîn gardîngni au run d'cadet.

Geraint Jennings
