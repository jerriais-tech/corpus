Les Pâraîsses dé Dgèrnésy

St. Martîn


Y'a un mênom pouor les cheins d'châque pâraîsse:

Les Villais sont les clyichards
Les St. Samsonnais sont les raînottes
Les Valais sont les hannetons
Les Cât'lains sont l's ânes pur sang
Les St. Saûveurais sont les freunmions
Les St. Pièrrais sont l's êtchèrbots
Les Tortévalais sont l's ânes à pid dé ch'va
Les Forêtains sont les bourdons
Les St. Andriais sont les craînchons
Les St. Martinnais sont les dravants
